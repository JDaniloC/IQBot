Versão 1.7 - Config UI and reconnect if error

            [RUN]
Para rodar, irá aparecer todos os comandos do bot:
        python bot.py -h
Para rodar diversas contas, crie um arquivo de configuração
para cada uma das contas, e coloque o nome dos arquivos nome
contas.txt sem nenhum espaço no final, depois: 
   - Windows
        powershell ./multi.ps1

            [ERROS COMUNS]
Se acontecer algum erro desses, apenas rode novamente:
 - FileNotFoundError
 - ssl.SSLEOFError
 - ssl.SSLError

            [CONFIG]
Caso quiser configurar via interface gráfica, rode o arquivo config.py!
Arquivo de configuração:
 - As opções estarão na forma de primeira|segunda, escolha uma ou outra.
 - Caso tiver valor-outro significa uma sequência de qualquer valor. E inf significa infinito.
O que significa:
 - email: email da conta, ou número
 - senha: senha da conta
 - tipo: se é a conta treino ou real
 - arquivo: nome do arquivo que vai estar as entradas
 - tipo_par: se a paridade é binaria/digital, se colocar auto, ele vai ver qual está disponível, ou tem maior profit
 - otc: se o tipo for binário e OTC
 - valor: o valor da entrada
 - tempo: o tempo da entrada
 - profit_minimo: porcentagem da retorno
 - correcao_entrada: segundos para se descontar do tempo de entrada
 - goal: valor objetivo que para se alcançar
 - soros: se vai realizar a sorosgale
 - percent_soros: valor de quanto irá aumentar na próxima soros, se 0 irá aumentar de acordo com o ganho
 - stoploss: valor limite de perdas
 - martin: se vai realizar martingale
 - max_gale: máximo de martingales
 - tipo_gale: caso haver gale, rode o script com -m
 - percent_martin: caso tipo_gale for "porcento", é o valor de em porcentagem (0, 50, 100...) que quer ganhar encima da perda

Exemplo:

[CONTA]
email = exemplo@email.com
senha = exemplo
tipo = treino|real

[ENTRADAS]
arquivo = entradas.txt
tipo_par = binaria|digital|auto
otc = True|False
valor = 1-inf
tempo = 1|5|15
profit_minimo = 0-100
correcao_entrada = 0-60

[WIN]
goal = 2-inf
soros = True|False
percent_soros = 0-inf

[LOSS]
stoploss = 1-inf
martin = True|False
percent_martin = 0-inf
max_gale = 1-inf
tipo_gale = seguro|simples|leve|agressivo|porcento
