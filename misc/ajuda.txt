            [RUN]

        python bot.py -h

            [CONFIG]
Caso quiser configurar via interface gráfica, rode o arquivo settings.py
Arquivo de configuração:
 - As opções estarão na forma de primeira|segunda, escolha uma ou outra.
 - Caso tiver valor-outro significa uma sequência de qualquer valor. E inf significa infinito.
O que significa:
 - email: email da conta, ou número
 - senha: senha da conta
 - tipo: se é a conta treino ou real
 - arquivo: nome do arquivo que vai estar as entradas
 - tipo_par: se a paridade é binaria/digital, se colocar auto, ele vai ver qual está disponível, ou tem maior profit
 - otc: se o tipo for binário e OTC
 - valor: o valor da entrada
 - tempo: o tempo da entrada
 - profit_minimo: porcentagem da retorno
 - tendencia: se vai querer negar entradas caso estiver contra a tendencia
 - correcao_entrada: segundos para se descontar do tempo de entrada
 - goal: valor objetivo que para se alcançar
 - soros: se vai realizar a sorosgale
 - stoploss: valor limite de perdas
 - martin: se vai realizar martingale
 - max_gale: máximo de martingales
 - tipo_gale: caso haver gale, rode o script com -m
 - delay: tempo para pegar o resultado

Exemplo:

[CONTA]
email = exemplo@email.com
senha = exemplo
tipo = treino|real

[ENTRADAS]
arquivo = entradas.txt
tipo_par = binaria|digital|auto
otc = True|False
valor = 1-inf
tempo = 1|5|15
profit_minimo = 0-100
tendencia = True|False

[WIN]
goal = 2-inf
soros = True|False

[LOSS]
stoploss = 1-inf
martin = True|False
max_gale = 1-inf
tipo_gale = seguro|simples|leve|agressivo|porcento|2.0-inf

[AJUSTES]
correcao_entrada = 0-inf
delay = -inf-inf
